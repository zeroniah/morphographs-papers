## Methodology

All papers use the **Systematic Evidence Integration Framework (SEIF)**, a 21-method triangulation protocol validated through AI-based inter-rater reliability analysis (ICC=0.971). The framework integrates:

- **Phonetic Analysis** (7 methods): Consonant root matching, sound change laws, phonotactic constraints
- **Semantic Analysis** (7 methods): Core meaning preservation, metaphorical mappings, polysemy patterns
- **Archaeological Evidence** (7 methods): Material artifacts, textual attestation, chronological depth
